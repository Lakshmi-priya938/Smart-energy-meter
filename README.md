# Smart Electric Meter Monitoring System

## Description
A smart system to monitor real-time energy consumption and transmit data wirelessly for efficient energy management.

## Features
- Real-time power monitoring
- Wireless data transmission
- Remote access via mobile/PC
- Energy usage tracking

## Components Used
- Microcontroller (Arduino/ESP8266)
- Current Sensor
- Voltage Sensor
- Wi-Fi Module

## Applications
- Smart homes
- Energy management systems
- Industrial monitoring
